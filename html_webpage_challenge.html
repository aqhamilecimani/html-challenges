Semantics in HTML5

A header

The header element specifies a header for a document or section. It should be used as a container for introductory content.You can have several header elements in one document.

A section

The section element defines a section in a document. According to W3C's HTML5 documentation: "A section is a thematic grouping of content, typically with a heading."A home page could normally be split into sections for introduction, content, and contact information.

A footer

The footer element specifies a footer for a document or section.This element should contain information about its containing element.A footer typically contains the author of the document, copyright information, links to terms of use, contact information, etc.You may have several footer elements in one document.
